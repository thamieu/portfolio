Pokedex

Ce projet est un projet qui permet de savoir les informations d'un pokemon via un appel d'une API.
On peut donc voir les point de vie, les attaques et on peut aussi rechercher un pokemon.
Ce projet m'a permis de comprendre comment fonctionne une API et comment faire une requête en python.
